2. Module-wise Standard Operating Procedures

Step-by-step guide for each module with inputs, outputs and responsible roles

2.1   Lead & Enquiry Management

📋
Logging a New Lead
Role: Sales
  1. Go to CRM & Sales → Leads & Enquiries
  2. Click + New Lead
  3. Enter: Client name, phone, email, location, requirement
  4. Select source (Referral / Walk-in / Social / etc.)
  5. Set initial status to New
  6. Save — lead code auto-generates (ENQ/001)
Input
Client contact, requirement details, source
Output
Lead record with auto-code ENQ/001
🔄
Follow-up & Conversion
Role: Sales
  1. Update lead status: New → Followed Up → Qualified
  2. Add follow-up notes and next action date
  3. If interested: create a Quotation from the lead
  4. If not interested: mark as Lost with reason
  5. Converted leads automatically link to the quotation
Statuses
New → Followed Up → Qualified → Converted / Lost
Output
Qualified lead → triggers Quotation creation

2.2   Quotation & BOQ Builder

📐
Creating a BOQ
Role: Sales / CEO
  1. Go to CRM & Sales → Quotations → Open BOQ
  2. Fill: Client name, site address, designer, validity
  3. Click + Add Room for each space (Bedroom, Kitchen…)
  4. Within each room, add line items: Description, Unit, Qty, Rate
  5. Amount auto-calculates per item; Grand Total updates live
  6. Fill Terms: Payment terms, warranty, scope includes/excludes
  7. Click Save — returns to Quotations list
Input
Room-wise items with qty & rates
Output
Quotation with auto ref STR/Q/001
👁
Preview & Send to Client
Role: Sales / CEO
  1. On the quotation list, click Preview
  2. Choose template: Classic / Modern / Minimal
  3. Review the formatted quotation document
  4. Click Print / PDF → Save as PDF
  5. Share PDF with client via email/WhatsApp
  6. Update status to Sent after sharing
Templates: Classic (dark gold), Modern (blue), Minimal (black & white). Template choice persists during the session.

2.3   Project Creation

🏗
Convert Quotation to Project
Role: CEO / Admin
  1. Set quotation status to Approved
  2. Click Convert to Project button on the quotation row
  3. System auto-creates: Project code (P001), Client name, Contract value
  4. All BOQ line items become Products under the project
  5. Each product carries the quoted price from the BOQ
  6. Project appears in Projects module with status: Planning
Input
Approved quotation
Output
Project P001 + products with quoted prices
📦
Adding Products Manually
Role: CEO / Factory Supervisor
  1. Go to Projects → Products
  2. Select the project from the filter dropdown
  3. Click + Add Product — project is pre-filled automatically
  4. Enter: Product name, category, estimated cost, notes
  5. Save — product linked to the current project
Categories: Wardrobe, Kitchen, TV Unit, Study, Cot/Bed, Shoe Rack, Loft, Pooja Unit, Dining, Bathroom Vanity, Other.

2.4   Bill of Materials (BOM) & Budget

📊
Setting Up Project BOM
Role: CEO / Factory Supervisor
  1. Go to Projects → Project BOM, select project
  2. Material BOM tab: Add each raw material — select from warehouse items or type custom. Enter Budget Qty and Waste %. Net Qty auto-calculates.
  3. Labour BOM tab: Add work centres (Carpentry, Site Installation, etc.). Enter hours and rate — amount auto-calculates.
  4. Other Costs tab: Enter Transport, Expenses, Contingency
  5. Milestones tab: Default 30/40/30 schedule pre-filled. Edit names, amounts, and triggers.
  6. Click Save BOM — all tabs saved as budget baseline
🎯
Product-level BOM
Role: CEO / Factory Supervisor
  1. Go to Projects → Products, click BOM on any product
  2. Add materials specific to this product (Wardrobe, Kitchen, etc.)
  3. Add labour operations and their costs
  4. Add product-specific expenses
  5. Live Margin shows: Quoted Price − BOM Cost (CEO/Admin only)
  6. Click Save Product BOM
Note: Margin and quoted price are visible to CEO and Admin only.

2.5   Milestones & Payment Schedule

💳
Milestone Management
Role: CEO / Accounts

Setting Milestones

  1. Open Project BOM → Milestones tab
  2. Default 30% / 40% / 30% schedule pre-populated
  3. Edit milestone names as needed
  4. Enter Amount or % — the other auto-calculates
  5. Set trigger for each milestone
  6. Save BOM — milestones are saved simultaneously

Milestone Triggers

TriggerWhat activates it
ManualYou click "Mark Ready"
On DeliveryAuto when delivery logged
On InstallationAuto when installation complete
On HandoverAuto when handover signed off
On QC PassAuto when QC inspection passes
Once a milestone status becomes Invoiced it is locked and cannot be edited or deleted.

2.6   Purchase Orders & Procurement

🛒
Raising a Purchase Order
Role: Procurement / CEO
  1. Go to Inventory → Purchase Orders → + New PO
  2. Select Vendor from vendor master
  3. Select Project and delivery destination (Warehouse / Site)
  4. Click + Add Item for each material: name, unit, qty, rate
  5. Amount auto-calculates per line; total updates live
  6. Set expected delivery date and any instructions
  7. Click Save — PO code auto-generates (PO/001)
  8. Preview and print PO using Preview / Print
  9. Change status to Sent to Vendor after sharing
📦
Receiving Goods
Role: Procurement / Factory Supervisor
  1. Open the PO (status must be Sent or Partially Received)
  2. Click Receive on the PO list row
  3. Enter received quantity per item
  4. If delivered to Warehouse: stock automatically updated
  5. PO status changes to Partially Received or Fully Received
  6. Record vendor bill in Finance → Vendor Bills
Important: All materials must arrive via PO or purchase to update warehouse stock. No ad-hoc stock additions.

2.7   Warehouse & Stock Management

🏪
Stock Operations
Role: Procurement / Factory Supervisor

Adding Stock

  1. Receive via PO (preferred) — qty auto-updates
  2. Or: Manual Purchase → set destination to Warehouse
  3. Or: Stock Adjustment for corrections

Stock Levels

  1. Go to Inventory → Warehouse
  2. View current qty, unit, rate per item
  3. Low stock items shown in Reports → Stock Report
  4. Reorder when stock falls below safety level

2.8   Material Requests & Issues

📋
Raising a Material Request
Role: Factory Supervisor
  1. Go to Inventory → Material Requests → + New Request
  2. Select Project, Product, and the required Item
  3. Enter quantity needed and reason
  4. System checks: Requested qty vs BOM Budget remaining
  5. If within budget → Auto-Approved, issue created instantly
  6. If over budget → status Pending, wait for CEO/Admin approval
Rule: No direct material issue. ALL issues must go through a Material Request first.
↩
Material Returns
Role: Factory Supervisor
  1. Go to Inventory → Material Returns → + Return
  2. Select the original Issue record
  3. Enter quantity being returned and reason
  4. Stock automatically added back to warehouse
  5. Remaining BOM budget qty is recalculated
Returns reduce issued qty and restore remaining budget, allowing future requests against the same BOM budget.

2.9   Manufacturing Jobs & Job Cards

🏭
Manufacturing Jobs
Role: Factory Supervisor
  1. Go to Factory / MES → Manufacturing Jobs
  2. Click Auto-Create Jobs to generate one job per product
  3. Or click + New Job to create manually
  4. Set priority (Normal / High / Urgent) and due date
  5. Open the job → click Operations to add work steps
  6. Assign each operation to a worker and machine
  7. Update operation status as work progresses
🪪
Job Cards for Workers
Role: Factory Supervisor
  1. Go to Factory / MES → Job Cards
  2. Filter by worker, status, or work centre
  3. Each card shows: Product, Operation, Machine, Due Date
  4. Click Print for Worker to print that worker's cards
  5. Click Print All for full factory card set
  6. Workers update their operation status (In Progress / Done)
Job cards include signature lines for worker acknowledgment.

2.10   Quality Control (QC)

✅
QC Inspection
Role: Factory Supervisor / CEO
  1. Go to Operations → QC Inspections
  2. Click + New QC Entry
  3. Select Project and Product being inspected
  4. Record inspection date and inspector name
  5. Set result: Pass or Fail
  6. Add remarks (defects, observations)
  7. If Pass → product cleared for delivery
  8. If Fail → return to factory for rework
A QC Pass on a product triggers any milestone set to On QC Pass.
📊
QC Reports
Role: CEO / Factory Supervisor
  1. Go to Operations → QC to view all inspections
  2. Filter by project or status (Pass/Fail)
  3. Track fail rate per project or product type
  4. Use data to improve material selection and carpentry SOP

2.11   Site Delivery & Installation

🚚
Site Delivery
Role: Site Supervisor
  1. Go to Site & Installation → Deliveries → + New Delivery
  2. Select project and list items being delivered
  3. Record delivery date, vehicle, and driver
  4. Log client acknowledgment
  5. Save — any milestone set to On Delivery auto-triggers
  6. Toast notification confirms milestone is "Ready to Invoice"
🔧
Installation & Handover
Role: Site Supervisor
  1. Go to Site & Installation → Installation
  2. Create installation record linked to project
  3. Assign installation team and set dates
  4. Update status: Scheduled → In Progress → Completed
  5. Enter handover date on completion
  6. Milestones for On Installation and On Handover auto-trigger
  7. Project status can be changed to Completed

2.12   Invoicing & Collections

🧾
Raising an Invoice
Role: Accounts / CEO
  1. Go to Finance → Milestones
  2. Find milestone with status Ready to Invoice
  3. Click Raise Invoice — invoice modal pre-fills amount & description
  4. Confirm details and Save — invoice code auto-generates (INV/001)
  5. Alternatively: go to Finance → Sales Invoices → + New Invoice
  6. Share invoice with client
💰
Recording Receipts
Role: Accounts / CEO
  1. Go to Finance → Receipts → + Record Receipt
  2. Select project and link to invoice
  3. Enter amount received, date, and mode (NEFT/Cash/Cheque)
  4. Save — outstanding balance auto-reduces
  5. View outstanding per project in Reports → Receivables
  6. Milestone status updates to Invoiced once fully received

5. Key Business Rules & Controls

System-enforced rules and important operating guidelines

Material Control Rules

RULE 1 — No Direct Material Issue: All material issues must go through a Material Request. No worker or supervisor can take stock directly from the warehouse without raising an MR first.
RULE 2 — Budget Enforcement: If the requested quantity exceeds the remaining BOM budget, the request cannot be auto-approved. It must be manually approved by CEO or Admin.
RULE 3 — Remaining Qty Calculation: Remaining = BOM Budget Qty − (Total Issued − Total Returned). This is recalculated live on every request.

Financial Controls

RULE 4 — Margin Confidentiality: Gross margin %, quoted price, and projected profit are visible only to CEO and Admin. All other roles see cost data but not the profit calculation.
RULE 5 — Milestone Locking: Once a milestone is invoiced, it cannot be edited or deleted. This ensures billing history integrity.
RULE 6 — Purchase to Warehouse: All material purchases must specify a destination (Warehouse or Site). Warehouse purchases automatically update stock levels.

Process Gates

Naming & Data Conventions

Project Codes are permanent once created. Never rename or delete a project code — it links to all products, BOMs, invoices, and material records.
Warehouse Items should use consistent naming. The same item with two different names will create duplicate stock entries.
Vendor Names must be set up in Master Data → Vendor Master before raising a PO against them.
Workers must be set up in Master Data → Workers before being assigned to job operations.
